# Session 4 — premade on-ramp

Known-good reference artifacts for anyone who missed Session 3, so they can drop in and continue without catch-up cost. Returning participants use their own Session 3 output instead.

This carries the Session 3 prototype work forward: three prototype directions, the flow through each, and a persona UX review that pressure-tests them.

## Contents

- `prototype-directions.md` — the three prototype directions (Signal, Decide, Shift), each with its Figma link, what it is, and how it answers the committed problem
- `app-flow.md` — the written screen-by-screen flow for each direction
- `ux-review.md` — the persona-based UX review with prioritized findings and triage
